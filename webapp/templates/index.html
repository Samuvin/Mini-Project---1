Multimodal Analysis

Three data modalities for comprehensive detection

Speech Analysis

Analyzes 22 acoustic features including jitter, shimmer, harmonics-to-noise ratio, and pitch variations to detect speech patterns associated with PD.

Handwriting Analysis

Evaluates handwriting features including pressure, velocity, acceleration, and tremor frequency to identify motor control patterns characteristic of PD.

Gait Analysis

Examines walking patterns including stride intervals, cadence, and gait variability to detect movement abnormalities typical in Parkinson's disease.

Technology Stack

Built on proven ML techniques and real clinical data

Machine Learning
  • Approach: Multimodal data fusion (early fusion)
  • Datasets: UCI Parkinson's, PaHaW, PhysioNet Gait
  • Features: ~42 combined features (speech + handwriting + gait)
  • Primary Model: Support Vector Machine (SVM)
  • Kernels: Linear, RBF, Polynomial, Sigmoid
  • Optimization: GridSearchCV with cross-validation
  • Class Balancing: SMOTE for robust training
Performance Metrics
  • Accuracy: 90-93% on test set
  • Precision: High true positive rate
  • Recall: Minimized false negatives
  • ROC-AUC: ~0.95 for optimized SVM
  • Validation: 5-fold cross-validation
  • Data: Real patient datasets only (no synthetic data)
