Multimodal Analysis

Three data modalities for comprehensive detection

Speech Analysis

Analyzes 22 acoustic features including jitter, shimmer, harmonics-to-noise ratio, and pitch variations to detect speech patterns associated with PD.

Handwriting Analysis

Evaluates handwriting features including pressure, velocity, acceleration, and tremor frequency to identify motor control patterns characteristic of PD.

Gait Analysis

Examines walking patterns including stride intervals, cadence, and gait variability to detect movement abnormalities typical in Parkinson's disease.

Technology Stack

Built on deep learning with dual attention and real clinical data

Deep Learning Architecture
  • Algorithm: SE-ResNet 1D CNN (He et al. 2016 + Hu et al. 2018)
  • Encoders: 3 per-modality SE-ResNet1D networks
  • Fusion: Attention-based modality weighting
  • Dual Attention: SE feature-level + modality-level
  • Explainability: Grad-CAM + SE weights + attention visualization
  • Framework: PyTorch with SMOTE class balancing
  • Datasets: UCI Parkinson's, PaHaW, PhysioNet Gait
Key Capabilities
  • 42 Features: Speech (22) + Handwriting (10) + Gait (10)
  • Residual Learning: Skip connections prevent vanishing gradients
  • Channel Attention: SE blocks learn feature importance
  • Modality Attention: Learns which input matters most per patient
  • Grad-CAM: Visual explanation of which features drove the prediction
  • Data: Real patient datasets only (no synthetic data)
