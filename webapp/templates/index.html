Multimodal Analysis

Three data modalities for comprehensive detection

Speech Analysis

Analyzes 22 acoustic features including jitter, shimmer, and harmonics-to-noise ratio to detect vocal patterns associated with PD.

Handwriting Analysis

Evaluates 10 handwriting features including pressure, velocity, and tremor frequency to identify motor control patterns.

Gait Analysis

Examines 10 walking parameters including stride intervals, cadence, and gait variability to detect movement abnormalities.

Under the Hood

Deep learning architecture with dual attention and real clinical data

Deep Learning Architecture
  • Algorithm: SE-ResNet 1D CNN
  • Encoders: 3 per-modality SE-ResNet networks
  • Fusion: Attention-based modality weighting
  • Dual Attention: SE feature + modality level
  • Explainability: Grad-CAM + SE + attention vis
  • Framework: PyTorch with SMOTE balancing
Key Capabilities
  • 42 Features: Speech (22) + Handwriting (10) + Gait (10)
  • Residual Learning: Skip connections for deep training
  • Channel Attention: SE blocks learn feature importance
  • Modality Attention: Dynamic per-patient weighting
  • Grad-CAM: Visual feature explanations
  • Real Data: UCI, PaHaW, PhysioNet datasets
