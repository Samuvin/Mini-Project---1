API Documentation

Complete reference for the PD Detection API

API Overview

The Parkinson's Disease Detection API provides endpoints for making predictions using trained machine learning models. All endpoints return JSON responses.

Base URL: http://localhost:5000/api

API Endpoints

GET /api/health

Description: Health check endpoint to verify API status.

Response:
{
  "status": "healthy",
  "model_loaded": true
}
POST /api/predict

Description: Make a prediction for a single sample.

Request (Combined Features):
{
  "features": [0.5, 1.2, -0.3, 0.8, ...]
}
Request (Multimodal):
{
  "speech_features": [0.5, 1.2, -0.3, ...],
  "handwriting_features": [0.8, 0.6, ...],
  "gait_features": [1.1, 0.9, ...]
}
Response:
{
  "success": true,
  "prediction": 1,
  "prediction_label": "Parkinson's Disease",
  "confidence": 0.89,
  "probabilities": {
    "healthy": 0.11,
    "parkinsons": 0.89
  }
}
POST /api/predict_batch

Description: Batch predictions for multiple samples.

Request:
{
  "features": [
    [0.5, 1.2, -0.3, ...],
    [0.8, 0.6, 0.1, ...],
    ...
  ]
}
Response:
{
  "success": true,
  "total_samples": 2,
  "results": [
    {
      "sample_id": 0,
      "prediction": 1,
      "prediction_label": "Parkinson's Disease",
      "confidence": 0.89,
      "probabilities": { "healthy": 0.11, "parkinsons": 0.89 }
    }
  ]
}
GET /api/model_info

Description: Get information about the loaded model.

Response:
{
  "success": true,
  "model_type": "SVC",
  "kernel": "rbf",
  "n_features": 42
}

Usage Examples

Python
import requests

url = "http://localhost:5000/api/predict"
data = { "features": [0.5, 1.2, -0.3, 0.8, ...] }

response = requests.post(url, json=data)
result = response.json()

print(f"Prediction: {result['prediction_label']}")
print(f"Confidence: {result['confidence']:.2%}")
cURL
curl -X POST http://localhost:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{ "features": [0.5, 1.2, -0.3, 0.8, ...] }'
JavaScript
fetch('http://localhost:5000/api/predict', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    features: [0.5, 1.2, -0.3, 0.8, ...]
  }),
})
.then(res => res.json())
.then(data => {
  console.log('Prediction:', data.prediction_label);
  console.log('Confidence:', data.confidence);
});

Error Codes

Code Description
200 Success — Request completed successfully
400 Bad Request — Invalid input data
401 Unauthorized — Missing or invalid JWT token
413 Payload Too Large — File size exceeds limit
500 Server Error — Model not loaded or processing error
